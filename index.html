Welcome to Guitar Sibling – guitar browser game with a real guitar!

Pick up a string you want to play on (yep, it's "unitar").
To enjoy Guitar Sibling you need to open this web page in Google Chrome. It's the only browser now that supports sound analysis on the client (yep, there is no fancy servers – all magic happens here).
To enjoy Guitar Sibling you need to update you Chrome browser to the latest version. Open Chrome settings and make sure you have version 58 or more (latest version supports sound analysis, Guitar Sibling works completely in browser).
